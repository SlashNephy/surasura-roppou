# テスト基盤とfixture Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Issue #10 の完了条件として、既存 Vitest/CI を前提に、後続 Issue が共有できる fixture とテストケースの置き場を整える。

**Architecture:** fixture は `src/test/fixtures/` に集約し、実装コードからは参照しない。e-Gov 取得層の既存テストは直書き fixture を module 化して実際の repository contract で使い続け、文字列正規化・条文参照パーサー・読みやすさ normalizer は後続 Issue が使う代表ケースを fixture として定義する。

**Tech Stack:** TypeScript 6, Vitest, Testing Library/jsdom, GitHub Actions, AGENTS.md の `playwright-cli open --headed` 方針。

```ts
import { describe, expect, it } from "vitest";

import { lawReferenceParseFixtures } from "./lawReferences";

describe("law reference parser fixtures", () => {
  it("keeps each parser fixture named and structurally coherent", () => {
    const names = new Set<string>();

    lawReferenceParseFixtures.forEach((fixture) => {
      expect(fixture.name).not.toBe("");
      expect(names.has(fixture.name)).toBe(false);
      names.add(fixture.name);
      expect(fixture.input).not.toBe("");
      expect(fixture.expected.confidenceFloor).toBeGreaterThanOrEqual(0);
      expect(fixture.expected.confidenceFloor).toBeLessThanOrEqual(1);

      if (fixture.expected.item !== undefined) {
        expect(fixture.expected.article).toBeDefined();
      }

      if (fixture.expected.paragraph !== undefined) {
        expect(fixture.expected.article).toBeDefined();
      }
    });
  });
});
```

```ts
import { describe, expect, it } from "vitest";
```
