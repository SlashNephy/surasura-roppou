# 条文ナビゲーション Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `/laws/:lawId` と `/laws/:lawId/articles/:article` で、目次、条文 anchor、条番号ジャンプ、mobile 目次を使って目的の条文へ移動できる本文ビューアを実装する。

**Architecture:** `core/viewer` に目次生成と表示、本文 anchor 表示を閉じ込め、`app` は route params、URL 遷移、スクロール、入力エラーだけを扱う。#14 の fixture-backed document をそのまま使い、永続化や e-Gov 接続は導入しない。目次クリックと条番号ジャンプは URL を正とし、URL 変更後に本文 anchor へスクロールする。

**Tech Stack:** React 19, TanStack Router, TypeScript 6, Tailwind CSS 4, Vitest, Testing Library, lucide-react, shadcn/ui style primitives。

```ts
import type { LawNode, LawNodeType } from "@/core/domain";

const tocNodeTypes = new Set<LawNodeType>([
  "Part",
  "Chapter",
  "Section",
  "Subsection",
  "Division",
  "Article",
  "SupplementaryProvision",
  "AppdxTable",
  "AppdxStyle",
]);

export interface LawTocItem {
  id: string;
  title: string;
  type: LawNodeType;
  depth: number;
  articleNumber?: string;
  children: LawTocItem[];
}

export const articleAnchorId = (articleNumber: string): string => `article-${articleNumber}`;

export const buildLawTableOfContents = (nodes: LawNode[]): LawTocItem[] => {
  const nodeById = new Map(nodes.map((node) => [node.id, node]));
  const topLevelNodes = nodes.filter((node) => node.parentId === undefined);

  return topLevelNodes.flatMap((node) => buildTocItems(node, nodeById, 1));
};

const buildTocItems = (
  node: LawNode,
  nodeById: Map<string, LawNode>,
  depth: number,
): LawTocItem[] => {
  const children = node.children
    .map((childId) => nodeById.get(childId))
    .filter((child): child is LawNode => child !== undefined);
  const childItems = children.flatMap((child) => buildTocItems(child, nodeById, depth + 1));

  if (!tocNodeTypes.has(node.type)) {
    return childItems;
  }

  const title = node.title ?? node.number ?? node.path;
  const articleNumber = node.type === "Article" ? node.number : undefined;

  return [
    {
      id: node.id,
      title,
      type: node.type,
      depth,
      ...(articleNumber === undefined ? {} : { articleNumber }),
      children: childItems,
    },
  ];
};
```

```tsx
import type { LawTocItem } from "./lawToc";

interface LawTableOfContentsProps {
  items: LawTocItem[];
  activeArticleNumber?: string;
  onSelectArticle: (articleNumber: string) => void;
}

export const LawTableOfContents = ({
  activeArticleNumber,
  items,
  onSelectArticle,
}: LawTableOfContentsProps) => (
  <nav aria-label="法令目次" className="grid gap-3">
    <h2 className="text-sm font-semibold text-foreground">目次</h2>
    {items.length === 0 ? (
      <p className="text-sm leading-6 text-muted-foreground">目次を表示できません</p>
    ) : (
      <ol className="grid gap-1 text-sm">
        {items.map((item) => (
          <LawTocItemRow
            key={item.id}
            activeArticleNumber={activeArticleNumber}
            item={item}
            onSelectArticle={onSelectArticle}
          />
        ))}
      </ol>
    )}
  </nav>
);

const LawTocItemRow = ({
  activeArticleNumber,
  item,
  onSelectArticle,
}: {
  activeArticleNumber?: string;
  item: LawTocItem;
  onSelectArticle: (articleNumber: string) => void;
}) => {
  const isActive = item.articleNumber !== undefined && item.articleNumber === activeArticleNumber;
  const paddingLeft = `${Math.max(item.depth - 1, 0) * 0.75}rem`;

  return (
    <li className="grid gap-1">
      {item.articleNumber === undefined ? (
        <div className="rounded-md px-2 py-1.5 font-medium text-foreground" style={{ paddingLeft }}>
          {item.title}
        </div>
      ) : (
        <button
          type="button"
          aria-current={isActive ? "location" : undefined}
          className="rounded-md px-2 py-1.5 text-left text-muted-foreground transition-colors hover:bg-accent hover:text-foreground aria-current:bg-accent aria-current:text-foreground"
          style={{ paddingLeft }}
          onClick={() => onSelectArticle(item.articleNumber)}
        >
          {item.title}
        </button>
      )}
      {item.children.length > 0 ? (
        <ol className="grid gap-1">
          {item.children.map((child) => (
            <LawTocItemRow
              key={child.id}
              activeArticleNumber={activeArticleNumber}
              item={child}
              onSelectArticle={onSelectArticle}
            />
          ))}
        </ol>
      ) : null}
    </li>
  );
};
```
