# 構造見出しと目次の数字変換 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 「読みやすい表示」で編、章、節、款、目の番号を本文とデスクトップおよびモバイル目次で算用数字へ変換し、「原文表示」で原文へ戻す。

**Architecture:** 原文の `LawNode` と `LawTocItem` は変更せず、既存の可読性変換を法令構造単位へ拡張する。目次コンポーネントは `displayMode` を受け取り、本文と同じ `applyLawTextDisplayMode` を描画時に適用する。法令ビューワー画面は一つの表示モードを本文と二つの目次へ渡す。

**Tech Stack:** React 19、TypeScript 6、Vitest、Testing Library、playwright-cli、mise、pnpm

---

- [x] **Step 3: 法令構造単位を変換する最小実装を書く**

`readability.ts` では既存の条項号用正規表現を維持し、一般語を避ける法令構造単位用正規表現を追加する。

```typescript
const articleNumberRegex = new RegExp(`第(${kanjiNumberPattern})(条|項|号)`, "g");
// 構造単位が「目標」や「編成」の語頭として現れる場合を除外しつつ、「第一目から」のように助詞が続く参照は変換する。
const structuralHeadingNumberRegex = new RegExp(
  `第(${kanjiNumberPattern})(編|章|節|款|目)(?!\\p{Script=Han})`,
  "gu",
);
```

変換関数を対象範囲に合わせて改名し、既存の別表と枝番処理を保持する。

```typescript
const transformLegalNumbers = (text: string): string =>
  text
    .replace(structuralHeadingNumberRegex, (_match, kanjiNumber: string, suffix: string) => {
      return `第${replaceKanjiNumber(kanjiNumber)}${suffix}`;
    })
    .replace(articleNumberRegex, (_match, kanjiNumber: string, suffix: string) => {
      return `第${replaceKanjiNumber(kanjiNumber)}${suffix}`;
    })
    .replace(appendixTableNumberRegex, (_match, prefix: string, tableNumber: string) => {
      return `${prefix}${replaceKanjiNumber(tableNumber)}`;
    })
    .replace(branchNumberRegex, (_match, prefix: string, branchNumbers: string) => {
      return `${prefix}の${branchNumbers.split("の").map(replaceKanjiNumber).join("の")}`;
    });
```

`transformReadableText` の `article-number` と `all` から `transformLegalNumbers` を呼ぶ。

```typescript
case "article-number":
  return transformLegalNumbers(text);
```

```typescript
case "all":
  return transformLegalNumbers(
    transformDates(transformLawNumbers(transformParentheses(text))),
  );
```
